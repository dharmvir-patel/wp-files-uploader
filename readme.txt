Adds ability to upload a file using dropzoneJs to the WordPress.

== Description ==

This plugin will let you upload, a file using dropzoneJs to the WordPress with a simple shortcode.

== Installation ==

Best is to install directly from WordPress. If manual installation is required, please make sure that the plugin files are in a folder named "wp-files-uploader" (not two nested folders) in the WordPress plugins folder, usually "wp-content/plugins".
